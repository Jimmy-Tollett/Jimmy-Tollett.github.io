CalPal

Collaborated on a development team for a class project to create CalPal, a smart calendar application. Contributed to the iOS front-end development using Swift and SwiftUI, gaining hands-on experience with networking, UI/UX design, and team collaboration. The project involved multiple teams working on different aspects of the application to ensure seamless integration. Although progress slowed during the summer, we remain committed to looking into continuing development and preparing the app for launch.

Guess The Line

For several years, my cousin has hosted a "Guess the Line" contest. This year, I took the initiative to enhance the user experience for both participants and my cousin by leveraging the Demos API. The project integrates the Demos API for graph visualization and input handling, while utilizing Google's Firebase for user authentication and submission management, streamlining the process of organizing and evaluating responses.
